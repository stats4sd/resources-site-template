# Multiple file uploads to trove show only one display name field

**Date Reported**: 2026-07-03
**Status**: Open
**GitHub Issue**: https://github.com/stats4sd/resources-site-template/issues/7

## Problem

When uploading multiple files to a trove, the interface only provides a single display name field. Users cannot set individual display names for each file being uploaded.

## Expected Behavior

Each file in a multi-file upload should have its own display name field, allowing users to customize the name for each file independently.
